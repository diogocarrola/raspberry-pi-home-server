# SmartTubeNext Installation (Quick)

1. Install `Downloader` from Play Store on Google TV.
2. Settings → Security → Allow `Downloader` to install apps (Unknown sources).
3. Open `Downloader` and enter:
   `https://github.com/yuliskov/SmartTubeNext/releases/latest/download/smarttube_next.apk`
4. Download and install the APK.
5. Open SmartTubeNext and optionally sign in.

Notes:
- If `Downloader` is not available, use `Send Files to TV` or copy the APK via USB.
